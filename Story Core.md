# Story Core

## 작성 원칙
- 이 문서는 사용자가 제공한 스토리 아이디어를 정리한 결과만 기록한다.
- 사용자의 명시적 요청이 없으면 에이전트가 임의로 스토리를 창작하지 않는다.
- 미확정 항목은 `[사용자 입력 대기]`로 표기한다.

## 1) 스토리 한 줄 요약
- [사용자 입력 대기]

## 2) 배경 세계관
- [사용자 입력 대기]

## 3) 플레이어 시작점
- [사용자 입력 대기]

## 4) 핵심 감정 테마
- [사용자 입력 대기]

## 5) 주요 인물
- [사용자 입력 대기]

## 6) 메인 스토리 구조
- [사용자 입력 대기]

## 7) 퀘스트 작성 규칙(스토리용)
- [사용자 입력 대기]

## 8) 중단/재개 친화 장치
- [사용자 입력 대기]
